# swipejobs-assignment

REST API to find top 3 most suitable jobs for workers


### Prerequisities

Make sure Node.js is installed in your computer

Fetch Dependencies: 
```
npm install
```
Run the application: 
```
node index.js
```

### API Doc


Entry point of REST API
```
localhost:5000
```

Find top 3 job for worker
```
GET /api/worker/jobs/{workerId}
```
